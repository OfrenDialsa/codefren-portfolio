# About This Project

This portfolio was built for speed and effortless customization—because I know we'd all rather be coding than wrestling with a CSS file for five hours. Leveraging the latest React ecosystem, it delivers a buttery-smooth user experience that’s fast enough to keep even the most impatient recruiter happy.

You can deploy it on **Netlify** or **Vercel**.

# Tech Stack

- **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **UI Components:** [Shadcn/UI](https://ui.shadcn.com/) & [Magic UI](https://magicui.design/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Package Manager:** [pnpm](https://pnpm.io/)

# Features

- **Easy Setup:** Configure the entire website through a single file: `src/data/resume.tsx`.
- **Responsive:** Optimized layout for all devices, including mobile, tablet, and desktop.
- **Performance:** Excellent Lighthouse scores, fully optimized for Next.js and Vercel.
- **Modern UI:** Interactive and smooth components powered by Magic UI and Framer Motion.
